Ağıllı Kitabxana İdarəetmə Sistemi
Layihə Haqqında
Bu proqram kitabxana işçilərinə oxucu məlumatlarını və kitabın qaytarılma müddətini asanlıqla izləmək üçün imkan yaradır. İstifadəçi məlumatları daxil edir, proqram isə bu gündən etibarən 15 günlük müddəti hesablayaraq bütün məlumatları bir QR-koda həkk edir.
 Əsas Funksiyalari
 Oxucunun adı, telefonu, emaili və kitabın adın istifadəçidən alır.
Kitabın qaytarılma tarixinin (15 gün sonra) sistem tərəfindən hesablanır.
Bütün məlumatların PNG formatında şəklidə saxlanılır.
Yaradılmış kodu telefon kamerası ilə oxudaraq məlumatları dərhal görmək imkanı əldə olunur.
Lazım Olan Kitabxanalar
