# kitabxana-qr-sistemi
Ağıllı Kitabxana İdarəetmə Sistemi Layihə Haqqında Bu proqram kitabxana işçilərinə oxucu məlumatlarını və kitabın qaytarılma müddətini asanlıqla izləmək üçün imkan yaradır. İstifadəçi məlumatları daxil edir, proqram isə bu gündən etibarən 15 günlük müddəti hesablayaraq bütün məlumatları bir QR-koda həkk edir.
